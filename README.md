# atomman-gb

Grain-boundary generator with Atomman

## Installation

```shell
git clone git@github.com:lan496/atomman-gb.git
cd atomman-gb
conda create -n atomman-gb python=3.10 pip
conda activate atomman-gb
pip install .
```
